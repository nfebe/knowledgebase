# Knowledgebase

Personal knowledgebase for tracking the state of work across different projects and organizations.

## Structure

```
knowledgebase/
├── README.md
└── useblocks/          # useblocks organization
    └── ubinfra/        # Infrastructure-related notes
```

## How It Works

- Each organization or project gets its own folder.
- Sub-folders are created as needed per repo or topic.
- Documents capture issues encountered, how they were resolved, and key takeaways for future reference.
