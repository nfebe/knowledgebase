# Knowledgebase

Personal knowledgebase for tracking the state of work across different projects and organizations.

## Structure

```
knowledgebase/
├── README.md           # this index
├── claude/             # Claude Code config (CLAUDE.md + commands/) mirrored from the live machine
├── mac/                # machine setup notes
├── nextcloud/          # Nextcloud server contribution notes
└── useblocks/          # useblocks organization
    ├── billing/        # invoice-grade time logs only
    ├── ubinfra/        # infrastructure notes
    └── ubtrace/        # ubTrace product notes
```

This Structure tree is the index: it lists every top-level org and major folder. Keep it current whenever one is added.

## How It Works

- Each organization or project gets its own folder.
- Sub-folders are created as needed per repo or topic.
- Documents capture issues encountered, how they were resolved, and key takeaways for future reference.
